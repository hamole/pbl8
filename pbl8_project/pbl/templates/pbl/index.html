PBL 9 - Alternative and Complimentry Medicines

This is a small site built as research for PBL 9 on the topic of alternative and complimentry medicines. You can select from different medicines using the drop down menu at the top of the page.

Each page contains a basic description of the medicine, some links to studies for and against the practice, a chart showing the interest in the medicine over time(based on searches) and the interest in the medicine based on region

At the bottom of each page is a form allowing discussants to add their comments to the page. You can either leave a general comment or reply to an existing comment by clicking the reply link next to the existing comment.
